YouTube ASL Overlay Sync

Local transcription is built in

This version transcribes YouTube audio through the local Python backend at http://127.0.0.1:8000. No API key is required.

Start the backend with python main.py, then reload the extension and play a YouTube video.

What to expect

  1. Captions still take priority when they exist.
  2. When captions are missing, the overlay sends 2-second audio chunks to the local backend.
  3. The backend uses Vosk to transcribe audio without any external API key.
